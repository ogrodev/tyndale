## [0.1.0] - 2026-04-12

### Added

- CLI commands: `init`, `auth`, `extract`, `translate`, `validate`
- React components: `T`, `Var`, `Num`, `Currency`, `DateTime`, `Plural`
- React hooks: `useTranslation`, `useLocale`, `useChangeLocale`, `useDictionary`
- Next.js adapter: server/client providers, middleware, config helper
- Incremental translation with delta detection
- Dictionary extraction and translation support
- Locale aliases for variant locale codes
- RTL direction support via `useDirection` hook
