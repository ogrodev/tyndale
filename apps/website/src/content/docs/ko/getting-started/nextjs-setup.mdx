## 어댑터 설치

```bash
npm install tyndale-next
```

## 설정

<Steps>

1. **Next.js config를 업데이트하세요**

   ```js
   // next.config.mjs
   import { withTyndaleConfig } from 'tyndale-next/config';

   export default withTyndaleConfig({
     // your existing Next.js config
   });
   ```

2. **middleware를 추가하세요**

   ```ts
   // middleware.ts
   export { tyndaleMiddleware as middleware } from 'tyndale-next/middleware';
   ```

   이는 로캘 감지, URL 라우팅, 로캘 쿠키 설정을 처리합니다.

3. **레이아웃에 서버 provider를 추가하세요**

   ```tsx
   // app/[locale]/layout.tsx
   import { TyndaleServerProvider } from 'tyndale-next/server';

   export default async function RootLayout({
     children,
     params,
   }: {
     children: React.ReactNode;
     params: Promise<{ locale: string }>;
   }) {
     const { locale } = await params;

     return (
       <html lang={locale}>
         <body>
           <TyndaleServerProvider locale={locale}>
             {children}
           </TyndaleServerProvider>
         </body>
       </html>
     );
   }
   ```

</Steps>

## 작동 방식

- `withTyndaleConfig`는 Next.js config에 로캘 경로 rewrite를 추가합니다
- middleware는 사용자의 선호 로캘을 감지하고 그에 맞게 리디렉션합니다
- `TyndaleServerProvider`는 서버 사이드에서 번역을 로드하고 React context를 통해 모든 컴포넌트에서 사용할 수 있게 합니다
