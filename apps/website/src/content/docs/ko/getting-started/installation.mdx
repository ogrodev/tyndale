## 사전 요구 사항

- Node.js 20 이상
- React 18.2+ 또는 19+
- AI provider API 키 (Anthropic, OpenAI 등)

## 패키지 설치

```bash
npm install tyndale tyndale-react
```

Next.js 프로젝트의 경우 어댑터도 설치하세요:

```bash
npm install tyndale-next
```

## 프로젝트 초기화

```bash
npx tyndale init
```

이 명령은 프로젝트 루트에 합리적인 기본값이 포함된 `tyndale.config.json` 파일을 생성합니다.
