import { Steps } from '@astrojs/starlight/components';

<Steps>

1. **Tyndale 설치**

   ```bash
   npm install tyndale tyndale-react
   ```

2. **프로젝트 초기화**

   ```bash
   npx tyndale init
   ```

   기본 설정이 포함된 `tyndale.config.json` 파일이 생성됩니다.

3. **AI provider 인증**

   ```bash
   npx tyndale auth
   ```

   안내에 따라 API 키를 설정하세요 (Anthropic, OpenAI 등).

4. **번역 가능한 콘텐츠 감싸기**

   ```tsx
   import { T, useTranslation } from 'tyndale-react';

   export function Welcome() {
     const t = useTranslation();
     return (
       <div>
         <T>Welcome to our app!</T>
         <p>{t('You have {count} items', { count: 3 })}</p>
       </div>
     );
   }
   ```

5. **추출 및 번역**

   ```bash
   npx tyndale translate
   ```

   Tyndale가 소스 파일을 스캔하고, 번역 가능한 문자열을 추출한 뒤, 구성된 모든 로캘에 대한 번역을 생성합니다.

</Steps>

## 방금 무슨 일이 있었나요?

Tyndale가 JSX에서 `<T>` 컴포넌트와 `useTranslation()` 호출을 스캔하고, 영어 원문 텍스트를 추출해 AI provider로 번역을 보낸 다음, `output` 디렉터리에 로캘 파일을 작성했습니다.

키 파일이 필요 없습니다. JSON을 수동으로 편집할 필요도 없습니다. 이제 번역이 준비되었습니다.
