## `tyndale init`

새 Tyndale 프로젝트를 초기화합니다. 합리적인 기본값으로 `tyndale.config.json`을 생성합니다.

```bash
npx tyndale init
```

## `tyndale auth`

AI 제공자 자격 증명을 구성합니다.

```bash
npx tyndale auth
```

## `tyndale translate`

구성된 모든 로캘에 대해 번역 가능한 문자열을 추출하고 번역을 생성합니다.

```bash
npx tyndale translate [options]
```

| 플래그 | 설명 |
|------|-------------|
| `--locale <code>` | 특정 로캘만 번역 |
| `--force` | 변경된 항목뿐 아니라 모든 문자열을 다시 번역 |
| `--dry-run` | 파일을 쓰지 않고 무엇이 번역될지 표시 |
| `--token-budget <n>` | AI 배치당 토큰 예산 (기본값: 50000) |
| `--concurrency <n>` | 최대 병렬 번역 세션 수 (자동 감지) |

## `tyndale validate`

모든 로캘에 완전하고 형식이 올바른 번역이 있는지 확인합니다. AI 호출은 수행되지 않습니다.

```bash
npx tyndale validate
```

문제가 발견되면 코드 1로 종료합니다. CI 파이프라인에 이상적입니다.

## `tyndale extract`

번역은 수행하지 않고 번역 가능한 문자열만 추출합니다. AI 제공자에게 전송될 내용을 검토할 때 유용합니다.

```bash
npx tyndale extract
```
