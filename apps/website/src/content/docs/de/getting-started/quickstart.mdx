import { Steps } from '@astrojs/starlight/components';

<Steps>

1. **Installiere Tyndale**

   ```bash
   npm install tyndale tyndale-react
   ```

2. **Initialisiere dein Projekt**

   ```bash
   npx tyndale init
   ```

   Dadurch wird `tyndale.config.json` mit Standardeinstellungen erstellt.

3. **Authentifiziere dich bei deinem AI-Provider**

   ```bash
   npx tyndale auth
   ```

   Folge den Anweisungen, um deinen API-Schlüssel zu konfigurieren (Anthropic, OpenAI usw.).

4. **Umschließe übersetzbare Inhalte**

   ```tsx
   import { T, useTranslation } from 'tyndale-react';

   export function Welcome() {
     const t = useTranslation();
     return (
       <div>
         <T>Welcome to our app!</T>
         <p>{t('You have {count} items', { count: 3 })}</p>
       </div>
     );
   }
   ```

5. **Extrahiere und übersetze**

   ```bash
   npx tyndale translate
   ```

   Tyndale durchsucht deine Quelldateien, extrahiert übersetzbare Strings und generiert Übersetzungen für alle konfigurierten Locales.

</Steps>

## Was ist gerade passiert?

Tyndale hat dein JSX nach `<T>`-Komponenten und `useTranslation()`-Aufrufen durchsucht, den englischen Quelltext extrahiert, ihn zur Übersetzung an deinen AI-Provider gesendet und Locale-Dateien in dein `output`-Verzeichnis geschrieben.

Keine Key-Dateien. Keine manuelle JSON-Bearbeitung. Deine Übersetzungen sind bereit.
