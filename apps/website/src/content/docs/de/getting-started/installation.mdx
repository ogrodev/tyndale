## Voraussetzungen

- Node.js 20 oder höher
- React 18.2+ oder 19+
- Ein API-Schlüssel eines AI-Providers (Anthropic, OpenAI usw.)

## Pakete installieren

```bash
npm install tyndale tyndale-react
```

Für Next.js-Projekte installiere außerdem den Adapter:

```bash
npm install tyndale-next
```

## Dein Projekt initialisieren

```bash
npx tyndale init
```

Dadurch wird im Stammverzeichnis deines Projekts eine `tyndale.config.json` mit sinnvollen Standardwerten erstellt.
