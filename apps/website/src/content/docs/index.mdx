## Supported frameworks

### App translation

- React
- Vite + React
- Next.js
- Astro components (`.astro`)

### Documentation translation

- Starlight
- Docusaurus
- VitePress
- MkDocs
- Nextra

See [Astro setup](/getting-started/astro-setup), [Next.js setup](/getting-started/nextjs-setup), and [Translating Documentation](/guides/translate-docs) for the framework-specific steps.
