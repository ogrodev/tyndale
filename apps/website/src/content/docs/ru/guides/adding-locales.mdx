title: Добавление новых локалей
description: Как добавить новый целевой язык в ваш проект Tyndale.
---

## Добавьте локаль

Откройте `tyndale.config.json` и добавьте код локали в массив `locales`:

```json
{
  "defaultLocale": "en",
  "locales": ["es", "fr", "ja", "de"]
}
```

Затем выполните:

```bash
npx tyndale translate
```

Tyndale сгенерирует переводы для новой локали, оставив существующие переводы без изменений.

## Алиасы локалей

Сопоставьте коды вариантов с каноническими с помощью `localeAliases`:

```json
{
  "localeAliases": {
    "pt-BR": "pt",
    "zh-TW": "zh"
  }
}
```

## Поддержка RTL

Tyndale автоматически определяет RTL-локали (арабский, иврит и т. д.). `TyndaleServerProvider` устанавливает атрибут `dir` на HTML-элементе. Используйте вариант Tailwind `rtl:` для стилей, специфичных для RTL.
