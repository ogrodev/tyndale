import { Steps } from '@astrojs/starlight/components';

<Steps>

1. **Установите Tyndale**

   ```bash
   npm install tyndale tyndale-react
   ```

2. **Инициализируйте проект**

   ```bash
   npx tyndale init
   ```

   Это создаст `tyndale.config.json` с настройками по умолчанию.

3. **Пройдите аутентификацию у вашего AI provider**

   ```bash
   npx tyndale auth
   ```

   Следуйте подсказкам, чтобы настроить ваш API key (Anthropic, OpenAI и т. д.).

4. **Оберните переводимый контент**

   ```tsx
   import { T, useTranslation } from 'tyndale-react';

   export function Welcome() {
     const t = useTranslation();
     return (
       <div>
         <T>Welcome to our app!</T>
         <p>{t('You have {count} items', { count: 3 })}</p>
       </div>
     );
   }
   ```

5. **Извлеките строки и выполните перевод**

   ```bash
   npx tyndale translate
   ```

   Tyndale просканирует ваши исходные файлы, извлечет переводимые строки и сгенерирует переводы для всех настроенных локалей.

</Steps>

## Что только что произошло?

Tyndale просканировал ваш JSX на наличие компонентов `<T>` и вызовов `useTranslation()`, извлек исходный английский текст, отправил его вашему AI provider для перевода и записал файлы локалей в директорию `output`.

Без key-файлов. Без ручного редактирования JSON. Ваши переводы готовы.
