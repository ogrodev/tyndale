---

## Предварительные требования

- Node.js 20 или новее
- React 18.2+ или 19+
- API-ключ провайдера AI (Anthropic, OpenAI и т. д.)

## Установка пакетов

```bash
npm install tyndale tyndale-react
```

Для проектов Next.js также установите адаптер:

```bash
npm install tyndale-next
```

## Инициализация проекта

```bash
npx tyndale init
```

Это создаст `tyndale.config.json` в корне вашего проекта с разумными настройками по умолчанию.
