---
title: Installation
description: Install Tyndale and its companion packages.
---

## Prerequisites

- Node.js 20 or later
- React 18.2+ or 19+
- An AI provider API key (Anthropic, OpenAI, etc.)

## Install packages

```bash
npm install tyndale tyndale-react
```

For Next.js projects, also install the adapter:

```bash
npm install tyndale-next
```

## Initialize your project

```bash
npx tyndale init
```

This creates a `tyndale.config.json` at your project root with sensible defaults.
