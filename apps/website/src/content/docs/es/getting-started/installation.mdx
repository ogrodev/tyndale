title: Instalación
description: Instala Tyndale y sus paquetes complementarios.
---

## Requisitos previos

- Node.js 20 o posterior
- React 18.2+ o 19+
- Una clave de API de un proveedor de IA (Anthropic, OpenAI, etc.)

## Instala paquetes

```bash
npm install tyndale tyndale-react
```

Para proyectos de Next.js, instala también el adaptador:

```bash
npm install tyndale-next
```

## Inicializa tu proyecto

```bash
npx tyndale init
```

Esto crea un `tyndale.config.json` en la raíz de tu proyecto con valores predeterminados razonables.
