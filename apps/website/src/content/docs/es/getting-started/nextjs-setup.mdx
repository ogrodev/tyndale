## Instala el adaptador

```bash
npm install tyndale-next
```

## Configuración

<Steps>

1. **Actualiza tu configuración de Next.js**

   ```js
   // next.config.mjs
   import { withTyndaleConfig } from 'tyndale-next/config';

   export default withTyndaleConfig({
     // your existing Next.js config
   });
   ```

2. **Agrega el middleware**

   ```ts
   // middleware.ts
   export { tyndaleMiddleware as middleware } from 'tyndale-next/middleware';
   ```

   Esto se encarga de detectar la configuración regional, enrutar URLs y establecer la cookie de configuración regional.

3. **Agrega el proveedor del servidor a tu layout**

   ```tsx
   // app/[locale]/layout.tsx
   import { TyndaleServerProvider } from 'tyndale-next/server';

   export default async function RootLayout({
     children,
     params,
   }: {
     children: React.ReactNode;
     params: Promise<{ locale: string }>;
   }) {
     const { locale } = await params;

     return (
       <html lang={locale}>
         <body>
           <TyndaleServerProvider locale={locale}>
             {children}
           </TyndaleServerProvider>
         </body>
       </html>
     );
   }
   ```

</Steps>

## Cómo funciona

- `withTyndaleConfig` agrega reescrituras de rutas de configuración regional a tu configuración de Next.js
- El middleware detecta la configuración regional preferida del usuario y redirige en consecuencia
- `TyndaleServerProvider` carga las traducciones del lado del servidor y las pone a disposición de todos los componentes a través del contexto de React
