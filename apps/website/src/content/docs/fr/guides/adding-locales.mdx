---
title: Ajout de nouvelles langues
description: Comment ajouter une nouvelle langue cible à votre projet Tyndale.
---

## Ajouter une locale

Ouvrez `tyndale.config.json` et ajoutez le code de la locale au tableau `locales` :

```json
{
  "defaultLocale": "en",
  "locales": ["es", "fr", "ja", "de"]
}
```

Ensuite, exécutez :

```bash
npx tyndale translate
```

Tyndale générera les traductions pour la nouvelle locale tout en laissant les traductions existantes inchangées.

## Alias de locale

Associez les codes de variante aux codes canoniques avec `localeAliases` :

```json
{
  "localeAliases": {
    "pt-BR": "pt",
    "zh-TW": "zh"
  }
}
```

## Prise en charge du RTL

Tyndale détecte automatiquement les locales RTL (arabe, hébreu, etc.). Le `TyndaleServerProvider` définit l’attribut `dir` sur l’élément HTML. Utilisez la variante `rtl:` de Tailwind pour les styles spécifiques au RTL.
