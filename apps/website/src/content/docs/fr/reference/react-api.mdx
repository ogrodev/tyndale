## Composants

### `<T>`

Encapsule le contenu textuel traduisible. Prend en charge les variables et les composants imbriqués.

```tsx
import { T } from 'tyndale-react';

<T>Welcome to our app!</T>
<T>Hello, {name}!</T>
```

### `<Plural>`

Gère la pluralisation en fonction d’une valeur de comptage.

```tsx
import { Plural } from 'tyndale-react';

<Plural value={count} one="1 item" other="{count} items" />
```

### `<DateTime>`

Formate les dates selon la langue active.

```tsx
import { DateTime } from 'tyndale-react';

<DateTime value={new Date()} />
```

### `<Currency>`

Formate les montants monétaires selon la langue active.

```tsx
import { Currency } from 'tyndale-react';

<Currency value={49.99} currency="USD" />
```

## Hooks

| Hook | Retourne | Objectif |
|------|---------|--------|
| `useTranslation()` | `t(source, vars?)` | Traduire des chaînes de manière programmatique |
| `useLocale()` | `string` | Code de langue actuel |
| `useChangeLocale()` | `(locale: string) => void` | Changer de langue à l’exécution |
| `useDictionary()` | `d(key) => string` | Accéder aux traductions du dictionnaire |

## Fonctions

### `msg(text)`

Marque une chaîne comme traduisible en dehors d’un contexte de composant.

```tsx
import { msg } from 'tyndale-react';

const errorMessages = {
  notFound: msg('Page not found'),
  unauthorized: msg('Please sign in to continue'),
};
```

### `msgString(text)`

Marqueur retournant une chaîne pour les contextes non React (Astro, Node, TS simple). Fonctionne de manière identique à `msg()` pour l’extraction, mais renvoie une `string` simple au lieu d’un élément React.

```ts
import { msgString } from 'tyndale-react';

const labels = {
  greeting: msgString('Hello'),
  farewell: msgString('Goodbye'),
};
```
