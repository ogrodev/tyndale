---

## Prérequis

- Node.js 20 ou version ultérieure
- React 18.2+ ou 19+
- Une clé API d’un fournisseur d’IA (Anthropic, OpenAI, etc.)

## Installer les paquets

```bash
npm install tyndale-react
npm install -D tyndale
```

Pour les projets Next.js, installez également l’adaptateur :

```bash
npm install tyndale-next
```

## Initialiser votre projet

```bash
npx tyndale init
```

Cela crée un `tyndale.config.json` à la racine de votre projet avec des valeurs par défaut pertinentes.
