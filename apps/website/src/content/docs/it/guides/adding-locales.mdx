---
title: Aggiungere nuove locale
description: Come aggiungere una nuova lingua di destinazione al tuo progetto Tyndale.
---

## Aggiungere una locale

Apri `tyndale.config.json` e aggiungi il codice della locale all'array `locales`:

```json
{
  "defaultLocale": "en",
  "locales": ["es", "fr", "ja", "de"]
}
```

Poi esegui:

```bash
npx tyndale translate
```

Tyndale genererà le traduzioni per la nuova locale lasciando inalterate quelle esistenti.

## Alias delle locale

Mappa i codici variante a quelli canonici con `localeAliases`:

```json
{
  "localeAliases": {
    "pt-BR": "pt",
    "zh-TW": "zh"
  }
}
```

## Supporto RTL

Tyndale rileva automaticamente le locale RTL (arabo, ebraico, ecc.). `TyndaleServerProvider` imposta l'attributo `dir` sull'elemento HTML. Usa la variante `rtl:` di Tailwind per gli stili specifici RTL.
