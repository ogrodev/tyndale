## Installa l'adattatore

```bash
npm install tyndale-next
```

## Configurazione

<Steps>

1. **Aggiorna la configurazione di Next.js**

   ```js
   // next.config.mjs
   import { withTyndaleConfig } from 'tyndale-next/config';

   export default withTyndaleConfig({
     // your existing Next.js config
   });
   ```

2. **Aggiungi il middleware**

   ```ts
   // middleware.ts
   export { tyndaleMiddleware as middleware } from 'tyndale-next/middleware';
   ```

   Questo gestisce il rilevamento della lingua, il routing degli URL e l'impostazione del cookie della lingua.

3. **Aggiungi il provider server al tuo layout**

   ```tsx
   // app/[locale]/layout.tsx
   import { TyndaleServerProvider } from 'tyndale-next/server';

   export default async function RootLayout({
     children,
     params,
   }: {
     children: React.ReactNode;
     params: Promise<{ locale: string }>;
   }) {
     const { locale } = await params;

     return (
       <html lang={locale}>
         <body>
           <TyndaleServerProvider locale={locale}>
             {children}
           </TyndaleServerProvider>
         </body>
       </html>
     );
   }
   ```

</Steps>

## Come funziona

- `withTyndaleConfig` aggiunge riscritture dei percorsi delle lingue alla configurazione di Next.js
- Il middleware rileva la lingua preferita dell'utente ed esegue il reindirizzamento di conseguenza
- `TyndaleServerProvider` carica le traduzioni lato server e le rende disponibili a tutti i componenti tramite il contesto React
