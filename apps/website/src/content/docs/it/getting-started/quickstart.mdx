import { Steps } from '@astrojs/starlight/components';

<Steps>

1. **Installa Tyndale**

   ```bash
   npm install tyndale tyndale-react
   ```

2. **Inizializza il tuo progetto**

   ```bash
   npx tyndale init
   ```

   Questo crea `tyndale.config.json` con le impostazioni predefinite.

3. **Autenticati con il tuo provider AI**

   ```bash
   npx tyndale auth
   ```

   Segui le istruzioni per configurare la tua API key (Anthropic, OpenAI, ecc.).

4. **Racchiudi i contenuti traducibili**

   ```tsx
   import { T, useTranslation } from 'tyndale-react';

   export function Welcome() {
     const t = useTranslation();
     return (
       <div>
         <T>Welcome to our app!</T>
         <p>{t('You have {count} items', { count: 3 })}</p>
       </div>
     );
   }
   ```

5. **Estrai e traduci**

   ```bash
   npx tyndale translate
   ```

   Tyndale analizza i tuoi file sorgente, estrae le stringhe traducibili e genera traduzioni per tutte le lingue configurate.

</Steps>

## Cosa è appena successo?

Tyndale ha analizzato il tuo JSX alla ricerca di componenti `<T>` e chiamate a `useTranslation()`, ha estratto il testo sorgente in inglese, lo ha inviato al tuo provider AI per la traduzione e ha scritto i file di lingua nella tua directory `output`.

Nessun file di chiavi. Nessuna modifica manuale dei file JSON. Le tue traduzioni sono pronte.
