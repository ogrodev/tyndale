## Prerequisiti

- Node.js 20 o successivo
- React 18.2+ o 19+
- Una chiave API di un provider AI (Anthropic, OpenAI, ecc.)

## Installa i pacchetti

```bash
npm install tyndale-react
npm install -D tyndale
```

Per i progetti Next.js, installa anche l'adattatore:

```bash
npm install tyndale-next
```

## Inizializza il tuo progetto

```bash
npx tyndale init
```

Questo crea un `tyndale.config.json` nella root del tuo progetto con impostazioni predefinite sensate.
