## Componenti

### `<T>`

Avvolge contenuti testuali traducibili. Supporta variabili e componenti annidati.

```tsx
import { T } from 'tyndale-react';

<T>Welcome to our app!</T>
<T>Hello, {name}!</T>
```

### `<Plural>`

Gestisce la pluralizzazione in base a un valore di conteggio.

```tsx
import { Plural } from 'tyndale-react';

<Plural value={count} one="1 item" other="{count} items" />
```

### `<DateTime>`

Formatta le date in base alla lingua corrente.

```tsx
import { DateTime } from 'tyndale-react';

<DateTime value={new Date()} />
```

### `<Currency>`

Formatta i valori monetari in base alla lingua corrente.

```tsx
import { Currency } from 'tyndale-react';

<Currency value={49.99} currency="USD" />
```

## Hook

| Hook | Restituisce | Scopo |
|------|---------|--------|
| `useTranslation()` | `t(source, vars?)` | Tradurre stringhe programmaticamente |
| `useLocale()` | `string` | Codice locale corrente |
| `useChangeLocale()` | `(locale: string) => void` | Cambiare locale in fase di esecuzione |
| `useDictionary()` | `d(key) => string` | Accedere alle traduzioni del dizionario |

## Funzioni

### `msg(text)`

Contrassegna una stringa come traducibile al di fuori del contesto di un componente.

```tsx
import { msg } from 'tyndale-react';

const errorMessages = {
  notFound: msg('Page not found'),
  unauthorized: msg('Please sign in to continue'),
};
```

### `msgString(text)`

Marcatore che restituisce una stringa per contesti non React (Astro, Node, TS semplice). Funziona in modo identico a `msg()` per l'estrazione, ma restituisce una `string` semplice invece di un elemento React.

```ts
import { msgString } from 'tyndale-react';

const labels = {
  greeting: msgString('Hello'),
  farewell: msgString('Goodbye'),
};
```
