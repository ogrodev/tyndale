---

## 前置条件

- Node.js 20 或更高版本
- React 18.2+ 或 19+
- 一个 AI 提供商 API 密钥（Anthropic、OpenAI 等）

## 安装包

```bash
npm install tyndale tyndale-react
```

对于 Next.js 项目，还需要安装适配器：

```bash
npm install tyndale-next
```

## 初始化你的项目

```bash
npx tyndale init
```

这会在项目根目录创建一个 `tyndale.config.json`，并提供合理的默认配置。
