title: Instalação
description: Instale o Tyndale e seus pacotes complementares.
---

## Pré-requisitos

- Node.js 20 ou posterior
- React 18.2+ ou 19+
- Uma chave de API de provedor de IA (Anthropic, OpenAI, etc.)

## Instale os pacotes

```bash
npm install tyndale tyndale-react
```

Para projetos Next.js, instale também o adaptador:

```bash
npm install tyndale-next
```

## Inicialize seu projeto

```bash
npx tyndale init
```

Isso cria um `tyndale.config.json` na raiz do seu projeto com padrões sensatos.
