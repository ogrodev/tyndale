import { Steps } from '@astrojs/starlight/components';

<Steps>

1. **Instale o Tyndale**

   ```bash
   npm install tyndale tyndale-react
   ```

2. **Inicialize o seu projeto**

   ```bash
   npx tyndale init
   ```

   Isto cria `tyndale.config.json` com as configurações predefinidas.

3. **Autentique-se com o seu fornecedor de IA**

   ```bash
   npx tyndale auth
   ```

   Siga as instruções para configurar a sua chave de API (Anthropic, OpenAI, etc.).

4. **Envolva conteúdo traduzível**

   ```tsx
   import { T, useTranslation } from 'tyndale-react';

   export function Welcome() {
     const t = useTranslation();
     return (
       <div>
         <T>Welcome to our app!</T>
         <p>{t('You have {count} items', { count: 3 })}</p>
       </div>
     );
   }
   ```

5. **Extraia e traduza**

   ```bash
   npx tyndale translate
   ```

   O Tyndale analisa os seus ficheiros de código-fonte, extrai as strings traduzíveis e gera traduções para todos os locales configurados.

</Steps>

## O que acabou de acontecer?

O Tyndale analisou o seu JSX em busca de componentes `<T>` e chamadas `useTranslation()`, extraiu o texto-fonte em inglês, enviou-o para o seu fornecedor de IA para tradução e escreveu ficheiros de locale no seu diretório `output`.

Sem ficheiros de chaves. Sem edição manual de JSON. As suas traduções estão prontas.
